How to do HTML?

HTML is super easy to learn!
Just learn the tags and their attributes!
For example, to make a paragraph, use the tag p, if you want to break a line, use the tag!
To make a heading use the tag h1,h2,h3 etc.
To make a link use the tag a with the attribute href.
To make a button use the tag button.
To make an image use the tag img with the attribute src.
To make text bold use the tag strong.
To make text italic use the tag em.
To make a horizontal line use the tag hr.
To make a footer use the tag footer.
